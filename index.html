Automotive Embedded Controls Simulator

OpenChassis ECU Lab

A browser-based chassis control simulator for ABS, EPS, steering angle sensor plausibility, diagnostics, and CAN-style signals.

Project Objective

This project demonstrates simplified automotive ECU control logic using a visual simulator. The first version focuses on software simulation before moving towards a low-cost hardware bench.
