Automotive Embedded Controls Simulator

OpenChassis ECU Lab

Small browser based simulator for trying out ABS, EPS, steering angle sensor logic and diagnostic states.

Project Objective

This project is being built as a simple public demo for chassis control logic. The current version adds driver inputs and a basic vehicle speed model.
